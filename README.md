## 💻 Getting Started

### Installation
```
npm install
```
### Develop Mode(터미널 2개 사용)
```
npm run json-server
npm run start
```

## 🔧 Stack
- **Language**: TypeScript
- **Library & Framework** : React.js
- **Database** : db.json
- **Deploy**: Render, GitHub Pages
